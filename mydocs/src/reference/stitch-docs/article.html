Quick Start

Initialize your environment with a single command. TechDocs handles dependencies and environment variables automatically.

terminal // bash 
$ npm install @obsidian-labs/techdocs-core
# Initializing documentation engine...
$ techdocs init --project-name my-awesome-docs
# Setting up core architecture...
Success! Documentation live at: http://localhost:3000
